# 🌍 Tourfinco – AI Budget Travel Planner

Tourfinco is an AI-powered budget travel planning web application designed to help users plan affordable trips based on their destination, budget, and group size.

This application generates a complete travel plan including places to visit, food options, and accommodation — all optimized to fit within the user's budget.

---

## 🚀 Features

- 🧠 AI-based travel planning
- 💰 Budget optimization for trips
- 👨‍👩‍👧‍👦 Multi-user trip planning (group-based)
- 🗺️ Destination-based itinerary generation
- 🏨 Suggestions for stays, food, and attractions
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML5, CSS3, JavaScript

**Backend & Tools:**
- Node.js
- Vite
- Supabase (Database & Auth)
